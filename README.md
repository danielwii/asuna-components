## asuna-components

### Install

```
$ npm i asuna-components
```

### Components List

- AdvancedButton
  - Submitting button with pop-confirm and tooltip
  - Modal button embedded form
- Drawer
  - One layer
  - Two layer
- Living loading
- Form
- Uploader
- Loading animations
- Weather animations
  - Snowing
  - Sun shining

### Clone locally:

```bash
$ git clone git@github.com:danielwii/asuna-components.git
$ cd asuna-components
$ yarn
$ yarn storybook
```
